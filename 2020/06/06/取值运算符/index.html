取值运算符

取值运算符

“*“的几种用途

<1> 乘法运算符

int x = 1;
int y = 2;
int z = x*y;

<2> 定义指针类型

char x;
char* y;

<3>取值运算符

* + 指针类型的变量

int a = 1;
int* p = &a;
printf("%d",*p)

此时 *p 中的* 就是取值运算符。

探测 *指针类型 的类型

int*** a;
int**** b;
int* c;

// *(a) = int**
// *(b) = int***
// *(c) = int

总结：

*加指针类型 的类型是 指针类型减去一个* (取出来的值的类型 = 原类型减去一个*)

指针取值的两种方式

*(p+0) 和 p[0]可以相互转换

*() 与 []

*(p+i) = p[i]

*(*(p+i)+k) = p[i][k]

(\(*(p+i)+k)+m) = p[i][k][m]
