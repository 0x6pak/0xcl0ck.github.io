指针类型

什么是指针类型？

基本类型 指针类型
char x; chat* x;
short x; short* x;
int x; int* x;
float x; float* x;
double x; double* x;
Student x; Student* x;

总结：

  1. 带有*的变量类型的标准写法：变量类型* 变量名
  2. 任何类型都可以带*。加上*以后是新的类型，统称“指针类型”
  3. *可以是多个
  4. 指针类型只是一个数据类型

指针类型 赋值

char* x;
short* y;
int* z;
Student* s;

x = (char*)1;
y = (short*)2;
z = (int*)3;
s = (Student*)4;

//第二种
char*** x;
short*** y;
int*** z;
Student*** s;

x = (char***)1;
y = (short***)2;
z = (int***)3;
s = (Student***)4;

指针变量 宽度

char x;
short y;
int z;

以上数据类型：char 1个字节，short 2个字节，int 4个字节。那么 指针类型的宽度是多少呢？

实验：

char* x;
short* y;
int* z;

x = (char*)1;
y = (short*)2;
z = (int*)3;

将以上代码运行后查看反汇编代码（可以多加几个*）

可以看到 dword

总结：

指针类型的变量宽度永远是4字节，无论类型是什么(char short int)，无论有几个*(char******)

指针类型 自加和自减(++ / –)

我们都知道，基本数据类型都可以自加或者自减，都是加1 或者减1。

指针类型也可以自加或自减

那么指针类型在 ++ 或者 – 的时候 和普通类型有什么区别呢？

实验：

char**** a;
short**** b;
int**** c;
a = (char****)100;
b = (short****)100;
c = (int****)100;
a++;
b++;
c++;
printf("%d %d %d",a,b,c);

输出的结果为：104 104 104

那么 结论：只要是指针类型，自增和自减都是4 对吗？

在*的数量在一个以上时，确实对。但是如果只有1个*呢？

输出的结果为：101 102 104

总结：

带 * 类型的变量(指针类型)，++ 或 – 的数量是：去掉一个 * 后变量的宽度

可以复制上面实验的代码，改成2个*和1个* 查看输出的结果
